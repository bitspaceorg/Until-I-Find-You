**UIFY** (Until I Find You) is a modular, latency-first motion-tracking engine written in Rust. It produces time-indexed streams of geometric primitives — points, bounding boxes, 6DoF poses, contours, face landmarks — and ships them to other programs via OSC / MIDI / shared memory / C ABI, with a first-class CLAP plugin for sample-accurate DAW automation.
